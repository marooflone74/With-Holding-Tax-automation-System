Withholding Tax Automation System (C++)

A console-based tax calculation system built according to Pakistan FBR Tax Year 2024 regulations.

🚀 Highlights

Implements multiple withholding tax sections (149, 150, 231AB, 235, 236A)

Progressive salary tax slab calculation

Dynamic filer vs non-filer tax logic

Strong input validation and error handling

Professional formatted console UI

Persistent file handling with timestamped records

Modular function-based architecture

Cross-platform console support (Windows/Linux)
